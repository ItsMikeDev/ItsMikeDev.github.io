CoralReports

A simple and customizable report plugin

About

CoralReports is a highly customizable report GUI for Minecraft Spigot. The plugin allows staff to easily manage reports and provide them with an array of information about the report.

Features

Feature Description
Report GUI A inventory GUI for reporting, fully customizable in the config.
Report Manager A inventory report manager that displays information about the report selected.
Report Notifications Notify your online staff when a report happens.
Report Files Stores each report in a file to prevent loss when reloading or restarting.

Commands and Permissions

Command Permission Description
/Report coralreports.report Gives the ability to report another player on the server.
/Reports coralreports.reports Gives the ability to view player reports in inventory gui.
coralreports.notify Will notify the player when someone is reported.
